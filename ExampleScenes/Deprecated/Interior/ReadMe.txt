This folder contains a free Blender model from Nextwave Multimedia.

The model is in interior.zip, which was downloaded on on 17 December, 2012, from the Nextwave web Multimedia site at:
  http://www.nextwavemultimedia.com/html/3dblendermodel.html

Nextwave Multimedia gave the RenderToolbox3 team permission to use nextwave free Blender models, for testing and scientific research, by email, on 12 December, 2012.

Thanks, nextwave!
