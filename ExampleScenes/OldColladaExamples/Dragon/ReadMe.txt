This folder contains a free Dragon model from Stanford University and MrBlueSummers.com.

The original model came from stanford_dragon.zip, which was downloaded on 7 February 2013 from mrbluesummers.com at:
  http://www.mrbluesummers.com/wp-content/uploads/2010/07/stanford_dragon.zip

The model was incorporated into the Dragon scene as part of Dragon.blend and Dragon.dae.

stanford_dragon.zip contained text file with this note:
This file brought to you by the geniuses at Stanford and MrBluesummers.com!
